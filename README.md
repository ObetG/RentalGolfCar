# RENTAL GOLF CAR �️

### Premium Golf Car Rental Solution

Website resmi **RENTAL GOLF CAR** — penyedia penyewaan mobil golf terpercaya di Indonesia. Menyediakan layanan rental mobil golf berkualitas, perawatan, sewa jangka panjang, dan customization untuk resort, hotel, event, dan komunitas golf.


---

## ✨ Features

- **Layanan Lengkap** — Rental mobil golf, service, sewa jangka panjang, dan customization
- **Responsive Design** — Tampilan optimal di semua device (desktop, tablet, mobile)
- **Fast Performance** — Dibangun dengan Vite untuk loading time yang cepat
- **Modern UI/UX** — Design clean dengan animasi smooth menggunakan Motion
- **SEO Optimized** — Lengkap dengan Open Graph, Twitter Cards, dan sitemap
- **PWA Ready** — Dapat diinstall sebagai Progressive Web App

## 🚀 Installation

### Prerequisites
- Node.js 18+ terinstall di sistem Anda

### Langkah Instalasi

```bash
# 1. Clone repository
git clone https://github.com/ObetG/RentalGolfCar.git
cd RentalGolfCar

# 2. Install dependencies
npm install

# 3. Setup environment variable
# Buat file .env.local dan tambahkan:
# GEMINI_API_KEY=your_api_key_here

# 4. Jalankan development server
npm run dev
```

Aplikasi akan berjalan di `http://localhost:3000`
